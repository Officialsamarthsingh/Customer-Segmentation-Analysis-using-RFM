# Customer Segmentation Analysis Using RFM

## Project Overview
This project focuses on analyzing customer purchasing behavior using RFM (Recency, Frequency, Monetary) analysis and customer segmentation techniques. The objective of the project is to identify different customer groups, understand customer behavior patterns, and generate business insights that can help improve customer retention, marketing strategies, and revenue growth.

The analysis was performed using a large-scale e-commerce transaction dataset containing customer demographics, product purchases, payment methods, churn information, and revenue details.

---

# Business Problem
Businesses often struggle to understand customer behavior and identify valuable customers. Without proper segmentation, all customers are treated similarly, leading to inefficient marketing strategies and reduced customer retention.

This project aims to:
- Identify high-value and loyal customers
- Detect inactive and churn-risk customers
- Analyze purchasing behavior across different customer groups
- Support data-driven business decisions

---

# Dataset Information

Dataset contains:

- Product category information
- Purchase transactions

- Customer activity details

- 250,000+ transaction records

# Project Workflow

## 1. Data Cleaning
- Handled missing values
- Removed duplicate records
- Converted date columns into datetime format
- Performed feature engineering

## 2. Exploratory Data Analysis (EDA)
Performed analysis on:
- Revenue distribution
- Product category performance
- Monthly revenue trends
- Payment method usage
- Customer demographics
- Churn analysis

## 3. Customer Metrics
Created customer-level metrics such as:
- Total spending
- Purchase frequency
- Average order value
- Customer activity trends

## 4. RFM Analysis
Segmented customers based on:
- Recency
- Frequency
- Monetary value

Customer segments include:
- Champions
- Loyal Customers
- Potential Customers
- At-Risk Customers
- Lost Customers

## 5. Dashboard Development
Built an interactive Power BI dashboard to visualize:
- Customer segments
- Revenue trends
- Churn analysis
- Business KPIs
- Customer behavior insights

---

# Key Insights

- A small percentage of customers contributes a major share of total revenue.
- Frequent customers generate higher long-term business value.
- Certain product categories perform significantly better than others.
- Some customer groups show high churn risk due to inactivity.
- Customer purchasing behavior varies across age groups and demographics.

---

# Business Recommendations

- Improve retention strategies for high-value customers.
- Re-engage inactive customers through personalized campaigns.
- Focus marketing efforts on high-performing product categories.
- Use customer segmentation for targeted marketing.
- Increase customer loyalty through rewards and repeat purchase incentives.
